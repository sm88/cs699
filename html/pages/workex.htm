The very brief details of the modules I worked for are as follows:

Module Name Description Responsibility
Phrases It was a small administration based functionality that allowed users to create text with rich text formatting (RTF) and then save it for future use in the main project. Implementation in JAVA, Test cases
Intervene Involved adding some new operations to existing interface and the corresponding implementation accross the domain. Design Document, Implementation in JAVA, Test cases
Plan Enhancement to an extension of Intervene allowing creation of multiple interventions Design Document, Implementation in JAVA, Test cases
Loader A sub-project which allows users to load data required for executing JUnit test cases when they are run on a new environment. It helped save hundreds of man-hours Implementation in JAVA
