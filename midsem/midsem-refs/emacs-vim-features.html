Cool and fast things you can do with emacs and vim

Emacs

  • My topmost reason: auto-complete; while editing text or tex or code, if there's a long word or variable name, just typing the first 2-3 characters and hitting alt-/ will complete it! This works nicely even when there are multiple possible completions.
  • Auto-indent while typing, or later: for Java, C, C++, Perl, Python, bash, LISP, tex, text, html ... the list is endless
  • Indent a selected region of code or text.
  • Comment a region of (latex, C, C++, Java, bash, python, ... anything)
  • Uncomment a region of ... anything
  • Integration with ispell to spell-check tex or text documents
  • Use of kill rings to easily paste something which was cut long back
  • Delete forward word, delete backward word
  • Delete forward sentence
  • Go to end or beginning of function definition, while editing code
  • Bibtex integration with emacs: can insert different types of bibtex entries with a single click
  • Cscope integration with emacs: keyboard shortcuts for all cscope functionalities
  • You can run emacs from within a terminal as well as using X-windows: very useful for (slow) remote logins.
  • Regular expression search and replace, optionally only in a selected region, and also optionally with confirmation.
  • You can split a window into multiple edit windows, horizontally or vertically. You can even have multiple emacs windows. This is very useful if I want to simultaneously look at different parts of the same document (e.g. I want to look at a structure definition, which is there at the top of file f.c, in one window, while adding code to it in the bottom of f.c in another window)
  • Customizable colours and fonts.
  • Save desktop state to continue from where you left off last time.
  • Repeat a particular typing N times
  • Repeat the previous command
  • Built-in help
  • Define macros, execute macros any number of times
  • You can run a shell from within emacs!
  • Emacs is extensible; there are add-ons for almost anything: email, www browsing, debugger, games, doctor, ...
  • Has a binary mode: useful to view non-ASCII files if you want!

Vim

  • Integration with cscope.
  • Search and replace regular expressions, optionally for a set of line numbers; most sed based commands are supported.
  • Faster than emacs: useful for quick editing.
  • Can use from within a terminal or using X-windows: very useful in (slow) remote login sessions.
  • Visual mode of editing: useful for editing columns, like in a spreadsheet!
  • Repeat a particular typing N times
  • Repeat previous command
  • Auto/Smart-indent is possible. If you can configure the .vimrc file, it can be enabled by default. Else, use set ai in the command mode.
  • Intuitive commands for various functions like save, etc.
  • Built-in demo.
  • Customizable syntax highlighting (again you need to modify the .vimrc file).
  • I'm not as much a vim fan as much as I'm an emacs fan; others can add more here.
